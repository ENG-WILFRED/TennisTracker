🚀 HOW TO USE

INITIAL SETUP (5 minutes):
  1. npm run seed                      # Create base data
  2. node seed-courts.js               # Create courts
  3. node seed-coach-players.js        # Create relationships ⭐

ACCESS THE FEATURE:
  1. Start app: npm run dev
  2. Login as coach/manager
  3. Click "👨‍🏫 My Players" in sidebar
  4. Browse and manage players

MANAGE YOUR PLAYERS:
  1. See all managed players in "My Players" tab
  2. View their sessions and statistics
  3. Click Message to communicate
  4. Track their progress over time

RECRUIT NEW PLAYERS:
  1. Click "All Players" tab
  2. Search for available players
  3. Click Message to send coaching offer
  4. Track new relationships as they develop

🔧 TECHNICAL IMPLEMENTATION DETAILS

FRONTEND ARCHITECTURE:
  • React functional components with hooks
  • useState for tab management, search, loading states
  • Client-side filtering and sorting
  • CSS-in-JS inline styling (consistent with dashboard)
  • Toast notifications using react-hot-toast

BACKEND ENDPOINTS:
  • GET /api/organization/{orgId}/players
  • GET /api/organization/{orgId}/players?type=managed&coachId={id}

DATABASE RELATIONS:
  • CoachPlayerRelationship (primary for feature)

STATE MANAGEMENT:
